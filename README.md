## Описание программы
Графическое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц, фильтровать список по жанру и минимальному объёму, а также сохранять и загружать данные в формате JSON.

## Требования для запуска
- Python 3.8 или выше
- Встроенные библиотеки: `tkinter`, `json`, `os` (дополнительная установка не требуется)

## Как запустить
```bash
# 1. Скачайте репозиторий или клонируйте его
git clone https://github.com/[ваш-логин]/book-tracker.git
cd book-tracker

# 2. Запустите файл
python main.py
