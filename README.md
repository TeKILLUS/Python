## 📖 Описание программы

**Book Tracker** — это графическое приложение для учёта прочитанных книг. Программа позволяет:

-  Добавлять книги с указанием названия, автора, жанра и количества страниц
-  Фильтровать список по жанру и минимальному количеству страниц
-  Сохранять и загружать данные в формате JSON
-  Автоматически обновлять список доступных жанров

Приложение разработано на Python с использованием библиотеки tkinter для создания графического интерфейса.

---

## 🛠 Требования для запуска

- **Python 3.8** или выше
- **Встроенные библиотеки** (не требуют установки):
  - `tkinter` — для GUI
  - `json` — для работы с данными
  - `os` — для работы с файловой системой
